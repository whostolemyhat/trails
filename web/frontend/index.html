About

Create procedural art by altering the options. Seed can be any text - changing it will alter the generated image. Using the same seed and options will always create the same image. Density will try to add more starting points per leaf. Min leaf size controls how large the leaves containing starting points are - larger means fewer leaves in the image. Canvas size controls the size of the image.
